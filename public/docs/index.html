Introduction

Base URL: http://localhost

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Todos

Display a listing of the resource.

Example request:
curl --request GET \
    --get "http://localhost/api/v1/todos?status=completed&search=project&sortBy=created_at&order=desc" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "http://localhost/api/v1/todos"
);

const params = {
    "status": "completed",
    "search": "project",
    "sortBy": "created_at",
    "order": "desc",
};
Object.keys(params)
    .forEach(key => url.searchParams.append(key, params[key]));

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "GET",
    headers,
}).then(response => response.json());

Headers

Example: application/json

Query Parameters

status   string  optional  

Filter todos by status. Example: completed

Search todos by title or details. Example: project

sortBy   string  optional  

Sort todos by a field. Example: created_at

order   string  optional  

Sort order. Must be one of asc or desc. Example: desc

Store a newly created resource in storage.

Example request:
curl --request POST \
    "http://localhost/api/v1/todos" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"title\": \"Finish project\",
    \"details\": \"Complete API implementation\",
    \"status\": \"in progress\",
    \"user_id\": 1
}"

const url = new URL(
    "http://localhost/api/v1/todos"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "title": "Finish project",
    "details": "Complete API implementation",
    "status": "in progress",
    "user_id": 1
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());
